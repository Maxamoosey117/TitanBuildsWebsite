Destiny 2 Titan Builds

This guide is intended to provide some titan builds that I have found or think I would find most fun. Remember this when looking at these builds is that the main intent is to have a playstyle with them where the most fun is involved. Some builds like the bubble titan are fairly obvious, while others like the hammer boi may not be so obvious. Try them out and see which build makes you feel like a true crayon eating, fallen smacking, hunter despising titan.
